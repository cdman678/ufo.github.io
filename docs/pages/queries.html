Insert New Sighting Query

The query used to insert a new sighting into the Sighting table is: 

INSERT INTO Sighting
VALUES (80100, 2019-04-01 21:00, 60, 1 minute, 30.2672, -97.7431);
Update Sighting Description Query

The query used to update the description values for SID=80100 in the Description table is: 

UPDATE Description
SET Shape='star', Sighting_description='object looked like a shooting star crossing the sky'
WHERE SID=80100;
Update Sighting Location Query

The query used to update the location values for SID=80100 from the Location table is: 

UPDATE Location
SET latitude=30.2672, longitude=-97.7431, City='Austin', State='TX', Country='us'
WHERE SID=80100;
Delete Sighting Query

The query used to delete the sighting with SID=80100 from the Sighting table is: 

DELETE FROM Sighting
WHERE SID=80100;
