UFO Shape

The Query used to generate this word cloud is: 

SELECT shape
FROM description;

/*MYSQL specific way of exporting a query to a text file*/
INTO OUTFILE '/var/lib/mysql-files/shapes.txt'
FIELDS ENCLOSED BY ''
TERMINATED BY ';'
ESCAPED BY '"'
LINES TERMINATED BY '\r\n';
