Designing the Database

Normalization

The dataset was in 1NF form to begin with. Below are the steps that were used to bring the database to 3NF

starting table(SID, timestamp, city, state, country, shape, seconds, time-desc, sighting-desc, latitude, longitude)

Full Dependencies:
  • SID -> timestamp, Seconds, time-desc
  • SID -> latitude, longitude -> city, state, country
  • SID -> city, state, country
  • SID -> shape, sighting-desc
2NF: Sighting(SID, timestamp, Seconds, time-desc, latitude, longitude, city, state, country)
         Description(SID, shape, sighting-desc)

Removing Transitive Dependencies:
  • SID -> timestamp, Seconds, time-desc, latitude, longitude
  • latitude, longitude -> city, state, country
  • SID -> shape, sighting-desc

3NF: Sighting(SID, timestamp, Seconds, time-desc, latitude, longitude)
         Description(SID, shape, sighting-desc)
         Location(latitude, longitude, city, state, country)
